# Yarn Spinner Zed 扩展

为 [Zed](https://zed.dev) 提供 [Yarn Spinner](https://www.yarnspinner.dev) 语言支持。

原项目：[yuna0x0/zed-yarn-spinner](https://github.com/yuna0x0/zed-yarn-spinner)

> [!WARNING]
> 本扩展及其内置的 Tree-sitter 解析器仍处于早期开发阶段，部分功能可能尚不完整或存在问题。

## 运行要求

- Windows
- 已安装 [Node.js](https://nodejs.org)，并可通过 `PATH` 中的 `node` 命令调用
- 已安装 [.NET 9](https://dotnet.microsoft.com/download/dotnet/9.0)，并可通过 `PATH` 中的 `dotnet` 命令调用

Yarn Spinner 语言服务器随扩展一同提供，并直接从 Zed 的扩展安装目录加载，无需额外下载或解压。

## 开发

如需开发本扩展，请参阅 Zed 文档中的[扩展开发指南](https://zed.dev/docs/extensions/developing-extensions)。
